## [2769. Find the Maximum Achievable Number](https://leetcode.com/problems/find-the-maximum-achievable-number/)

### Approach I: 

#### Java:
```java
class Solution {
    public int theMaximumAchievableX(int num, int t) {
        int maxAchiveable = t * 2 + num;
        return maxAchiveable;
    }
}
```
