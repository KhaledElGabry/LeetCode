## [2413. Smallest Even Multiple](https://leetcode.com/problems/smallest-even-multiple/)

### Approach I: 

#### Java:
```java
class Solution {
    public int smallestEvenMultiple(int n) {
        if (n % 2 == 0) {
            return n;
        }
        return 2 * n;
    }
}
```
